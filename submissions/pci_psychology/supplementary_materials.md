## Table S1: GRM Item Parameters

Graded Response Model (GRM) item discrimination ($a_1$) and
threshold ($d_1$, $d_2$, $d_3$) parameters for the nine HPT
items. Parameters were estimated using the `mirt` R package
with a constrained multi-group model (High- vs.
Low-ideology groups defined by median split on the composite
FR-LF + KSA score). An all-others-as-anchor DIF testing
strategy with Bonferroni-corrected significance level
($\alpha$ = .006) detected no significant DIF for any item.
Because no items were flagged, the constrained model---in
which item parameters are held equal across groups---is the
final model. The parameters below therefore apply to both
ideology groups.

| Item  | Subscale          | $a_1$  | $d_1$  | $d_2$  | $d_3$  |
|-------|-------------------|--------|--------|--------|--------|
| CONT1 | Contextualisation |  1.864 |  2.812 |  0.718 | -1.964 |
| CONT2 | Contextualisation |  1.382 |  2.082 |  0.728 | -1.273 |
| CONT3 | Contextualisation |  1.364 |  2.763 |  0.768 | -1.324 |
| POP1  | Presentism (rev.) | -1.016 |  0.207 | -1.345 | -2.674 |
| POP2  | Presentism (rev.) | -0.355 |  0.903 | -0.843 | -2.217 |
| POP3  | Presentism (rev.) | -0.630 |  0.683 | -1.068 | -2.727 |
| ROA1  | Recognition       |  1.193 |  2.633 |  1.175 | -0.804 |
| ROA2  | Recognition       |  0.719 |  2.159 |  0.481 | -1.777 |
| ROA3  | Recognition       |  1.430 |  2.706 |  1.313 | -0.953 |

*Note.* $a_1$ = item discrimination; $d_1$, $d_2$, $d_3$ =
threshold (intercept) parameters on the latent scale. POP items
are reverse-scored at the response level so that higher scores
indicate more contextualised reasoning across all subscales.
Negative $a_1$ values for POP items reflect the reversed scoring
direction: these items discriminate in the opposite direction on
the latent trait compared to CONT and ROA items, consistent with
the three-factor CFA model reported in the manuscript. The
constrained model was retained because Bonferroni-corrected
likelihood-ratio DIF tests showed no significant difference
between free and constrained parameters for any item (all
$p$ > .01).
